Bilan de la première semaine

Lors de cette première semaine à BeCode, j'ai appris plusieurs choses (encore heureux !) très importantes pour la suite de la formation.

Les valeurs importantes chez BeCode

  1. La solidarité/l'entraide
  2. Le respect
    1. Des autres
    2. Des règles de savoir-vivre
    3. De soi-même
  3. La débrouillardise
  4. Le dépassement de soi
  5. La remise en question
  6. Et bien d'autres !

Les commandes importantes à connaître pour naviguer dans le terminal

  • pwd
  • cd
  • ls
  • mkdir
  • touch
  • mv
  • cp

Les commandes GIT à connaître absolument

  • git init
  • git clone
  • git add
  • git commit -m ""
  • git push
  • git pull
  • git merge
  • git checkout

(Liste non-exhaustive)

Mon ressenti

J'avais énormément d'appréhension avant le premier jour mais elle s'est vite dissipée dès le premier jour.

Je pensais que j'aurais des difficultés à apprendre de façon autonome, et cela s'est révélé faux, en tout cas jusqu'à présent. L'ambiance chez BeCode est vraiment sympa et on s'entraide les uns les autres, permettant à chacun de progresser.

Bref, je suis vraiment satisfaite de mes débuts chez BeCode ! Et je me suis fait un nouvel ami : je vous présente Alfie le canard !
